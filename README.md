# Air602 - 5led

Blink five LEDs sequentially.

## Wiring

```
-------|
       |
    GND>---330 Ohm-----|
       |               |
    TX1>---LED white---|
       |               |
    RX1>---LED blue----|
       |               |
    RTS>---LED yellow--|
       |               |
    CTS>---LED green---|
       |               |
     IO>---LED red-----|
       |
-------|
```

## Effect

LEDs light up one by one, in the order of red - green - yellow - blue - white.
