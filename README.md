# Air602 - weather

Measure temperature and humidity with HTU21D sensor, and display on TM1637 four-digit LED display.

Air602 pin | W600 pin | HTU21D pin | TM1637 pin
-----------|----------|------------|-----------
IO         | PB08     | +          |
CTS        | PB08     |            | CLK
RTS        | PB08     |            | DIO
RX1        | PB08     | CL         |
TX1        | PB08     | DA         |
USB 5V     |          |            | 5V
GND        |          | -          | GND
